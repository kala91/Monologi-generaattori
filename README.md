# Dramaturgi: Monologi-LARP Generaattori

Yksinpeli-monologigeneraattori, joka ohjaa pelaajaa dramaturgisella taksonomialla jeepform-tyyliin.

This repository is preserved as part of the Elopeli prototype history. See [META_CONTEXT.md](META_CONTEXT.md) for the research context, findings, and lineage.

Original AI Studio app: https://ai.studio/apps/drive/16u1rUk1txBI6PgqGgVttSNYAyNyIK9UH

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set `GEMINI_API_KEY`.
3. Run the app:
   `npm run dev`
